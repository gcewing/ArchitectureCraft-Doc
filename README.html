ArchitectureCraft

ArchitectureCraft (formerly called Greg's Blocks) is a mod for the game Minecraft. It provides blocks for creating various architectural features that can't otherwise be achieved easily or at all. Currently included are blocks for building sloping roofs, pillars and rounded corners, and windows with frames.

Change History

Version 1.7.0
Improved compatibility with mods that mine blocks automatically.
Changed recipes for Hammer and Chisel to make them more distinctive and less likely to conflict with other mods.
Restored ability to use half-slabs as raw material. Will use two slabs for each batch of output.
Fixed: Mined shape blocks sometimes failed to stack with other blocks of the same type.
Version 1.6.0
Shape blocks now emit light if they are made from a light-emitting material. (Will not affect existing blocks created with an older version, they will need to be re-crafted and re-placed.)
Fixed: Shift-clicking result slot in sawbench spammed inventory with duplicate items.
Version 1.5.3
Shift-clicking now supported in sawbench.
Version 1.5.2
1.10.2: Fixed bug causing double material usage in sawbench.
1.10.2: Prevented non-cubical blocks from being used as raw material.
Version 1.5.1
1.10.2: Updated for Minecraft 1.10.2. (Thanks to Yakov Lipkovich for doing most of the work!)
Fixed smooth shading of pillars and cylinders.
Version 1.4.2
Fixed cladding turning into a full block when breaking a clad block.
Version 1.4.1
Fixed placement of banisters on ArchitectureCraft stairs and upside-down vanilla stairs.
1.8.9: Guard against "net.minecraft.block.BlockAir cannot be cast to gcewing.architecture.BaseMod$IBlock"
Version 1.4.0
Added stair and slab blocks.
Fixed rendering of cladding on transparent base materials.
1.7.10: Fixed orientation of projected textures on east/west faces.
1.8.9: Fixed inventory rendering of cladding made from some types of wooden logs.
Version 1.3.4
Glass, ice and other transparent blocks can now be used as raw material.
Version 1.3.3
1.7.10: Fixed crash caused by Chisel animated textures.
Version 1.3.2
1.7.10: Fixed index out of bounds in onTextureStitchEventPre when used with some other mods.
Version 1.3.1
Fixed orientation of roof blocks placed against existing upside-down roof blocks.
Backported to 1.7.10.
Version 1.3.0
Updated for Minecraft 1.8.8/1.8.9.
Version 1.2.1
Fixed server errors concerning collision boxes.
Version 1.2.0
New blocks: Railings for balconies and stairways.
Version 1.1.0
New blocks: Bevelled and rounded corner caps, arch-building parts
Version 1.0.1
Fixed crash when breaking shaped blocks made from some vanilla materials.
Fixed a texture corruption bug.
Version 1.0.0
Initial version for Minecraft 1.8, based on Greg's Blocks.
